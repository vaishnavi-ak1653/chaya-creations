# Chaya Creations — Fashion Website

A responsive editorial website for **Chaya Creations**, a fashion brand.

Open `index.html` in any browser — no build step needed.

## What's included

- Full-page hero with animated text
- Collections grid with hover states
- About / brand story section
- Chiguru sub-brand feature section
- Lookbook gallery
- Testimonials
- Contact section

## Design decisions

Built around a warm cream-and-sage palette with serif/sans-serif contrast to communicate timelessness and approachability. Every section has its own copy voice — aspirational in the hero, conversational in about, editorial in the lookbook.

## Stack

Plain HTML · CSS custom properties · Vanilla JS
